# 手機測試流程

不用 Android Studio 也可以測試 APK。

## 安裝

1. 執行打包：

```powershell
.\scripts\build-debug-apk.ps1
```

2. 產生 APK：

```text
D:\projects\TTt\app\build\outputs\apk\debug\app-debug.apk
```

3. 將 APK 安裝到手機。

## 基本測試

1. 打開「點按翻譯助手」。
2. 設定來源語言為「繁體中文」，目標語言為「簡體中文」。
3. 翻譯尾碼填入 `(13`，並確認「尾碼前自動加入空格」已勾選。
4. 點「儲存翻譯設定」。
5. 點「啟動懸浮窗」並允許顯示在其他應用程式上層。
6. 收起懸浮窗後，在小輸入列輸入 `這個真的很好看`。
7. 按確認。
8. 應顯示「已翻譯並複製」，輸入框會清空。
9. 到任一文字輸入欄手動貼上，應出現 `这个真的很好看 (13`。

## 點按輔助測試

1. 打開系統「無障礙」設定。
2. 點「已安裝的應用程式」。
3. 開啟「點按翻譯助手點按輔助」。
4. 回到 App，點「啟動懸浮窗」。
5. 拖曳綠色準心到要點按的位置。
6. 收起懸浮窗後，按小列上的綠色 `▶`。
7. 應每 400ms 點按一次，狀態顯示剩餘秒數。
8. 按紅色 `■` 可停止；90 秒到也會自動停止。
9. 若點進文字輸入欄，點按應停止。

若系統顯示拒絕授予無障礙存取權，先在 App 主畫面點「開啟系統應用程式列表」，選「點按翻譯助手」→右上角「⋮」→「允許受限制設定」，再回到「無障礙」→「已安裝的應用程式」開啟。

## 多國語測試

多國語翻譯需要先在主畫面填入可用的 `/v1/translate` 後端。

例如：

```text
https://你的後端/v1/translate
```

未設定後端時，繁體中文/簡體中文以外的方向會提示需要翻譯後端。
